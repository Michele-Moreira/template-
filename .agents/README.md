# .agents

Diretório para instruções operacionais do projeto voltadas a agentes.

## Arquivos

- `rules/` - regras gerais do repositório e critérios de edição.
- `skills/` - quando usar skills e como selecionar a mais adequada.
- `subagents/` - orientação para dividir tarefas e acionar subagentes.

## Objetivo

Manter em um único lugar as instruções que ajudam agentes a trabalhar neste template sem espalhar contexto por vários arquivos.
