# Skills recomendadas

Use skills quando a tarefa pedir um fluxo especializado ou uma extração estruturada.

## Situações comuns

- Pesquisa e extração na web: usar uma skill de scraping, busca, crawl ou pesquisa profunda.
- Estruturação de conhecimento: usar uma skill de knowledge base quando o resultado precisar virar base consultável.
- QA de páginas e fluxos: usar uma skill de teste/interação em navegador.
- Design de site: usar uma skill de análise de design quando a tarefa for reproduzir linguagem visual.

## Critério prático

- Se a tarefa depende de conteúdo externo, escolha uma skill antes de improvisar.
- Se a tarefa envolve várias páginas, prefira skill de crawl ou pesquisa multi-etapa.
- Se a tarefa pede um artefato final organizado, prefira a skill que já entregue esse formato.
- Mantenha os exemplos e as orientações genéricos, sem citar ferramentas ou fornecedores específicos quando isso não for necessário.
