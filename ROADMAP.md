# Roadmap

Este documento ajuda a organizar ideias e prioridades para projetos rápidos, estudos e portfólio.

## Visão geral

- Objetivo principal: entregar valor rápido com código reutilizável.
- Público-alvo: hackathons, projetos de estudo, demonstrações de portfólio.

## Prioridades

1. MVP funcional
2. Estrutura clara e documentação
3. Código legível e fácil de estender
4. Deploy ou demonstração visual

## Itens de curto prazo

- Configuração inicial de projeto
- Documentação de uso
- Estrutura de pastas para desenvolvimento e testes

## Itens de médio prazo

- Adição de exemplos reais
- Scripts de automação simples
- Melhorias no estilo e nas boas práticas

## Itens de longo prazo

- Templates para diferentes stacks
- Documentação de padrões reutilizáveis
- Integração com deploys e portfólios
